# Interactive Computer Quiz and Game in Python

This repository contains two beginner‑friendly Python scripts designed to test and reinforce knowledge of computer fundamentals. Both scripts are interactive, track correct and incorrect answers, and provide performance feedback at the end. They are ideal for students, educators, or anyone interested in practicing computer literacy through Python programming.

---

## 📂 Files Included
- **choose-quiz.py**  
  A multiple‑choice quiz covering essential computer science concepts such as hardware, software, memory, and networking.

- **computer-quiz.py**  
  An interactive game combining open‑ended and True/False questions to strengthen understanding of computer basics.

---

✨ Features
Beginner‑friendly Python code using input(), conditionals, and scoring logic.

Tracks both correct and incorrect answers.

Provides percentage scores at the end of each session.

Includes multiple‑choice, open‑ended, and True/False formats.

Easy to extend with new questions or categories.
